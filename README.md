# CV Website - Nguyễn Văn Hải | AI Agent Developer

Professional portfolio website built with pure HTML, CSS, and JavaScript.

## 🌐 Live Demo

[https://reiseto.github.io/cv-website/](https://reiseto.github.io/cv-website/)

## ✨ Features

- 🎨 Modern dark/light theme with smooth transitions
- ✨ Interactive particle background
- 📱 Fully responsive design
- 🎯 Smooth scroll navigation
- 📊 Animated stat counters
- 💎 Glassmorphism UI components
- ⚡ Zero dependencies, pure vanilla JS

## 🚀 Deployment

This site is deployed via **GitHub Pages**. Simply push to the `main` branch and enable GitHub Pages in repository settings.

## 📁 Structure

```
cv-website/
├── index.html    # Main HTML structure
├── style.css     # Styling with CSS variables & animations
├── script.js     # Interactive features & particle system
└── README.md     # This file
```

## 🛠️ Tech Stack

- HTML5 (Semantic)
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- Vanilla JavaScript (Canvas API, Intersection Observer)
- Google Fonts (Inter)
- Font Awesome Icons
